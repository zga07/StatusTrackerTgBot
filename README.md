# Order Status Tracker Bot

Универсальный бот на **Go** для отслеживания статусов заказов. Позволяет администраторам управлять базой заказов, а пользователям — мгновенно получать информацию о местонахождении посылки по трек-коду.

---

## Основные возможности

* **Для пользователей:**
    * Поиск заказа по уникальному трек-коду.
    * Просмотр полной хронологии перемещений (история статусов).
    * Автоматические уведомления при обновлении статуса (после первой проверки кода).
* **Для администратора:**
    * Создание новых заказов с автоматической генерацией кодов.
    * Обновление статуса заказа.

---

## Настройка бота с Docker

### 1. Настройка окружения
Создайте файл `.env` в корне проекта и заполните его своими данными:


BOT_TOKEN=botfather_token


ADMIN_ID=tg_id


DB_USER=admin


DB_PASSWORD=password


DB_NAME=db_name


DB_HOST=db


DB_PORT=5432


### 2. Запуск Docker
Запустите Docker и введите в терминале:

docker-compose up --build -d
